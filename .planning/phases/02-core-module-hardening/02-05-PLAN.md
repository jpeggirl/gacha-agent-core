  <behavior>
    DealScorer:
    1. score(listing, card, fmv) where fmv.prices has 3 items and score is 85
       -> signal: 'buy' (downgraded from strong_buy)
       -> reasoning includes "Thin market" warning

    2. score(listing, card, fmv) where fmv.prices has 3 items and score is 70
       -> signal: 'fair' (downgraded from buy)

    3. score(listing, card, fmv) where fmv.prices has 10 items and score is 85
       -> signal: 'strong_buy' (no downgrade — sufficient data)

    4. score(listing, card, fmv) where seller feedback < 95%
       -> signal: 'avoid' (hard filter, regardless of score or market depth)

    EbayScanner:
    5. scan(card, grade) when eBay API returns 401
       -> ScanResult with empty listings (not thrown error)
       AND console.error is called

    6. scan(card, grade) with valid API response
       -> ScanResult with parsed listings

    7. buildSearchQuery(card, grade) produces correct query string
  </behavior>
  <implementation>
    DealScorer changes:

    1. Pass `fmv` to `deriveSignal()` — it already receives it as `_fmv` (unused). Remove the underscore prefix:
       ```typescript
       private deriveSignal(score: number, listing: EbayListing, fmv: FairMarketValue): DealSignal {
       ```

    2. After the score-to-signal mapping, add thin-market downgrade:
       ```typescript
       // Thin market downgrade: < 5 price data points = unreliable FMV
       if (fmv.prices.length < 5) {
         if (signal === 'strong_buy') signal = 'buy';
         else if (signal === 'buy') signal = 'fair';
       }
       ```

    3. In `buildReasoning()`, add thin-market warning after population section:
       ```typescript
       if (fmv.prices.length < 5) {
         parts.push(`Thin market (${fmv.prices.length} price points) — FMV less reliable`);
       }
       ```

    EbayScanner changes:

    4. Wrap the body of `scan()` in try/catch:
       ```typescript
       async scan(card: ResolvedCard, grade?: number): Promise<ScanResult> {
         try {
           await this.ensureToken();
           const query = this.buildSearchQuery(card, grade);
           const listings = await this.searchListings(query, grade);
           return { card, listings, scannedAt: new Date().toISOString(), totalFound: listings.length };
         } catch (error) {
           const msg = error instanceof Error ? error.message : String(error);
           console.error(`[EbayScanner] Scan failed for ${card.name}: ${msg}`);
           return { card, listings: [], scannedAt: new Date().toISOString(), totalFound: 0 };
         }
       }
       ```

    - EbayScanner tests: Use MSW to mock eBay OAuth token endpoint and Browse API. Test success path, API error path, and query building.
  </implementation>
